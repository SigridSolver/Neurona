✓ Respuesta correcta: C (Vende productos a: P, R | Compra productos de: Ninguno)

Para determinar las relaciones comerciales del país $W$, analicemos la tabla principal por separado:

1. **¿A quién le vende productos el país W?**
Que el país $W$ le venda a otros países significa que esos otros países **compran** productos de $W$. Si revisamos la columna *"Compra productos de:"*, vemos que: * El país $P$ compra de: $S, V, P, \mathbf{W}$ (por lo tanto, $W$ le vende a $P$). * El país $R$ compra de: $R, V, \mathbf{W}, Q$ (por lo tanto, $W$ le vende a $R$). * De este modo, $W$ vende productos a: $\mathbf{P, R}$.

2. **¿De quién compra productos el país W?**
Que el país $W$ le compre a otros países significa que esos otros países le **venden** productos a $W$. Si revisamos la columna *"Vende productos a:"* para todos los países ($P, Q, R, S$), ninguno de ellos lista al país $W$. Por tanto, $W$ no realiza compras registradas. * De este modo, $W$ compra productos de: $\mathbf{Ninguno}$.
